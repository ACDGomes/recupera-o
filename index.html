Os Principais Comandos para Criar um Site

Comando HTML

O comando básico para criar uma estrutura de site em HTML é:

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título do Site</title>
</head>
<body>
    <h1>Título Principal</h1>
    <p>Conteúdo do site.</p>
</body>
</html>

Comando CSS

Para estilizar o site, você utiliza comandos CSS, como:

body {
    font-family: Arial, sans-serif;
}
h1 {
    color: blue;
    font-size: 30px;
}
h2 {
    color: red;
    font-size: 20px;
}
p {
    font-family: 'Times New Roman', serif;
    color: black;
    font-size: 15px;
}
